Sobre o To-do List

Este é um projeto de estudo para aprofundar meus conhecimentos em HTML, CSS, JavaScript e, principalmente, no framework Bootstrap versão 5.3.6. O objetivo principal é criar uma aplicação web simples e funcional de lista de tarefas (To-do List).

Tecnologias Utilizadas
  • HTML5
  • CSS3
  • JavaScript
  • Bootstrap 5.3.6
  • Google Fonts (Poppins)
  • Font Awesome
  • jQuery (para funcionalidades específicas)
Objetivos de Aprendizagem
  • Compreender a estrutura e os componentes do Bootstrap.
  • Utilizar os componentes visuais do Bootstrap (navbars, cards, modals, etc.).
  • Implementar interatividade com JavaScript e jQuery.
  • Organizar o código de forma modular e eficiente.
  • Criar uma interface de usuário responsiva e agradável.

Espero que este projeto possa demonstrar a aplicação prática das tecnologias mencionadas e o meu progresso no aprendizado de desenvolvimento web.
